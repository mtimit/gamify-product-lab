# Gamify Product Lab

Мини-игра по созданию проектов, которые приносят деньги.

## Как запустить

### ⚠️ ВАЖНО: Используйте локальный сервер!
Из-за ES6 модулей, приложение нужно запускать через веб-сервер, а не просто открывать файл.

### ✅ Вариант 1: Live Server (Рекомендуется)
1. Установите расширение "Live Server" в VS Code
2. Откройте `index.html`
3. Нажмите правой кнопкой мыши и выберите "Open with Live Server"
4. Приложение откроется в браузере автоматически

### ✅ Вариант 2: Python HTTP Server
```powershell
# В PowerShell
cd C:\Users\hollo\Documents\gamify-product-lab
python -m http.server 8000
```
Затем откройте http://localhost:8000 в браузере

### ❌ НЕ работает: Прямое открытие файла
Просто открыть `index.html` двойным кликом НЕ БУДЕТ работать из-за CORS политики браузера для модулей.

## Отладка

Если что-то не работает:
1. Откройте консоль браузера (F12)
2. Смотрите сообщения об ошибках
3. Читайте подробную инструкцию в **DEBUG.md**
4. Или откройте **test.html** для быстрой диагностики

## Функциональность

✅ UI из трёх экранов: Dashboard / Projects / Experiments
✅ Создание проектов и экспериментов
✅ Изменение статусов проектов
✅ Начисление XP по базовым правилам:
  - добавил идею (+10 XP)
  - перевёл проект по стадии (+20 XP)
  - завершил эксперимент (+50 XP)
  - добавил доход (1 XP за каждые $10)
✅ Рост уровня (level-up)
✅ Хранение в localStorage
✅ Ручное добавление дохода
✅ Достижения (achievements)
✅ Streak (streak дней активности)

## Структура проекта
